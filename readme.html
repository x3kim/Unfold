About Unfold

Purpose & Functionality

Unfold is a minimalist desktop utility designed to flatten a nested folder structure. It takes a selected folder, recursively finds every file within it and all its subdirectories, and copies them into a single, new folder.

This is useful for consolidating files from complex directory trees, for example, collecting all photos from a backup or gathering all documents from a project archive into one place.

Key Features

  • Safe Copying: The original folder and its files are never modified.
  • Conflict Resolution: If files with the same name exist in different subfolders, they are automatically renamed (e.g., [conflict-1]-file.txt) to prevent overwriting.
  • Detailed Logging: Every action (copy, rename, error) is documented in a documentation.md file in the output folder.
  • Modern UI: Includes a theme switcher, live progress updates, and a multi-language interface.
